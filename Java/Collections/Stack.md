# Stack

**Last In, First Out**

### Creating Stack

```Java
Stack<Integer> stk = new Stack<>();
```

### Push

Puts the variable into the stack and then returns in

```java
stk.push(2);
```

### Pop

Removes the top element and returns it

```java
stk.pop();
```

### Peek

Returns the last element

```java
stk.peek();
```

### IsEmpty

```java
stk.isEmpty();
```

### Size

```java
stk.size();
```
