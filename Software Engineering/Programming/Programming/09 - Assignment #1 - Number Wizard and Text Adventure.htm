software engineering and game design

Task #1 - number wizard

Number wizard is a guessing game. The object of the game is to guess the random number chosen. The game will have two rounds.

  • In the first round the computer will pick a random number between 1-1000. You will need to try and guess that number and the computer will keep track of the number of guesses you take. The computer should also tell you if the number is higher or lower than your guess.
  • In round two, you will select a number and enter it into at input field (to keep you honest) and the computer must try to guess your number. You will use key press events to tell the computer if the number is higher or lower.

After the two rounds there should be an output saying who took the least number of guesses as the winner.

 

task #2 - Prison escape

Prison escape is a text based choose your own adventure game. The game starts with you in a prison cell. The user is then given options as to how to proceed. The game should follow a story line and in the end, the prisoner escapes or is captured. Your story should have a minimum of 5 story lines (see example below).

This example should not be viewed as "absolute", it is just an example to give you an idea of how to organize the story and make five layers.

assets

You will need to find appropriate assets to include for your games. Creativity is key, but make sure they make sense. Again, I recommend using pictures with the file format of .png If you have Draw or any other graphics program, you can change the format with that.
