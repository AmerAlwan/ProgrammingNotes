TEJ 3MI Grade 11 Computer Engineering Technology

software engineering and game design

Zodiac adventures

Your job is to continue with the program that we started to develop in class together. Your program should ask the user for their name and their date of birth. The program should then output their age, some information about their zodiac sign, and should cause a very simple graphical animation to occur.

Success criteria

  • I can create a fully functional program with the specific requirements
  • I can document my code to include internal comments
  • I can use ALL concepts covered so far (variables, input, output, selection, repetition, assets, and functions)
  • I can problem solve
  • I can complete and submit my assignment on time
