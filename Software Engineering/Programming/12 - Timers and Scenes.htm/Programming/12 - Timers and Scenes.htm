You will notice that we have two variables for our timer. The first variable is a float, which is necessary as the countdown will occur based on screen rate, not actual time. This means that the time will be presented as a very long decimal value. The second variable is a rounded value for the time so that the user sees what appears to be a normal countdown clock. It is important that we keep the float variable because if we were just to use a rounded value we would lose 1/2 seconds repeatedly. We will also start by assigning a value of 10 to our timer variable as this is what we will count down from.
