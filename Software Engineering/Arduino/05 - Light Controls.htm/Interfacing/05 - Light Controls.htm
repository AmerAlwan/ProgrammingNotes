Light Controls

For your first challenge you need to use the skills that you have learned so far to create two programs (each programs will run separately but the wiring may be very similar).

Program #1 - Speed Count

In this program you will use the potentiometer to control the speed at which a 7-segment display counts from zero to nine.

Program #2 - Controlled Count

In this program you will use the potentiometer to change the numbers from zero to nine on the 7-segment display. At the same time the RGB LED will change colour with the change of each number (this change should be sharp and not gradual).
