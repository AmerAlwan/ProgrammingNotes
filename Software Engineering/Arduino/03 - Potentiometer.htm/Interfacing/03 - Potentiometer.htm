potentiometer

A potentiometer is actually a variable resistor, which means that when the knob is turned from low to high the amount of resistance increases. In this circuit you will learn how to control the brightness of an LED.

Wiring Setup

Use the diagrams below to setup your Arduino and breadboard.

You will need an Arduino, breadboard, one LED, one potentiometer, one 220Ω resistor, and jumper wires.

Exercise

  1. Add the code "Serial.begin(9600);" in the void setup method. Add the code "Serial.print(sensorValue);" at the bottom of the loop. Upload your program. Go to Tools --> Serial Monitor. What happens?
  2. Wire up three to five led's and use the potentiometer to control which led lights up.
