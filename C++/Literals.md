# Literals

### Binary

```cpp
int b = 0b01111; //15
```

### Octal

Is a compressed Binary (Based on group of 3  binary digits -> 2^3)

```c++
int o = 010; //8
```

### Hexadecimals

Is a compressed binary (based on group of 4 binary digits -> 2^4)  **Ex:** 1011 0011 1010 111 = B3AF

```cpp
int h = 0xf; //15
```
